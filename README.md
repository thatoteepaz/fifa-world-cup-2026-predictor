# fifa-world-cup-2026-predictor
This application analyzes two national teams and predicts the most likely winner by calculating a performance score from several key football metrics.

## Features
- FIFA Ranking Analysis
- Recent Form Evaluation
- Goals Scored & Conceded
- Head-to-Head Records
- Squad Depth Analysis
- Injury Assessment

## Technologies
- Python 3
